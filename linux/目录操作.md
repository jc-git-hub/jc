## 目录操作

```shell
ls --help  # 查看帮助
man ls # 查看帮助

-a  # 显示所有文件  包含隐藏文件
-l  #列出文件或目录的详细信息

-h   # 用最佳的阅读体验
```

### 目录的创建及删除

```shell
mkdir 目录名  
mkdir -p  递归创建目录
mkdir -p a/{b,c}/{d,e,f}

rm -rf 目录名  #r 递归删除  f强制
# 删除之前要小心
# rm -rf /*  删除所有文件
```

## 文件的操作

```shell
# 创建文件
vim 文件名  #一定要保存
touch  文件名  #创建文件

# 查看文件的内容
cat 文件名	  # 从上往下输出内容
tac 文件名  # 从下往上输出内容
head -n 3 文件名 #显示该文件的前三行
tail -n 3 文件名  #显示文件后三行
watch -d -n 秒数 cat 文件名  # 每秒刷新一次结果  ctrl+c结束
more 文件名 #分页显示文件  #空格翻页  回车  换行 q退出
less 文件名  #也是分页  g首页  G尾页
b 向前翻页  空格 向后翻页  q退出
stat  文件名  #文件详细信息

```

### 往文件中写入内容

```shell
echo '内容' >文件名	覆盖
echo '内容'>>文件名 追加
```

### 文件的删除

```shell
rm -i #询问是否删除  y是n否
rm -f #强制删除
rm -r  #递归删除

rm -rf #删除当前目录所有  不要再/使用
rm -rf *.txt  # 删除当前目录下  .txt后缀的文件

```

### 文件的拷贝


### 文件查找

```shell
find [路径] [参数] [文件名]
-name 根据文件名查找
-iname 根据文件名查找  文件名不区分大小写
-mtime +/-n -n表示 n天  +n 超过n天的文件    -n n天以内修改的
-user  根据所属用户
find / -user 'root'
-size  [+/-]n[c/k/m/g] -size +3c  大小大于3个字节的
-perm  根据权限查找
-maxdepth  按照深度进行查找
-ls  查出来的结果按照列表的形式展示
find / -maxdepth 1 -size -10k -ls
# 查找/目录下 深度是1大小再10k以下 的文件

```
